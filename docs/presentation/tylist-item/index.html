    TYListItem 列表项

    TYListItem 是一个列表项组件。

    代码演示

    基础列表

    import { TYListItem, IconFont } from 'tuya-panel-kit'
    

    <TYListItem
      theme={{
        fontColor: '#333333',
        subFontColor: '#999',
        descFontColor: 'red',
        cellBg: 'white',
      }}
      title={"标题"}
      subTitle={"副标题"}
      Action={<IconFont color="rgba(0, 0, 0, 0.2)" name="arrow" />}
      onPress={() => console.log(111)}
    />

    文案过长适配

    import { TYListItem } from 'tuya-panel-kit'
    

    <TYListItem
      theme={{
        fontColor: '#333333',
        subFontColor: '#999',
        descFontColor: 'red',
        cellBg: 'white',
      }}
      title={"主标题超过一行规定的宽度时，文案将自行换行"}
      subTitle={"副标题"}
      arrow={true}
      arrowColor="#fff"
      onPress={() => console.log(111)}
      Action={<IconFont color="rgba(0, 0, 0, 0.2)" name="arrow" />}
    />
