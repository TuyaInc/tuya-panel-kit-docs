    SwitchButton 开关

    SwitchButton 是一个开关组件。

    代码演示

    基础样式一

    const [value1, setValue1] = React.useState(true);
    const [value2, setValue2] = React.useState(false);
    

    <View>
      <SwitchButton
        value={value1}
        onValueChange={v => setValue1(v)}
        style={{ marginRight: 14 }}
      />
      <SwitchButton value={value2} onValueChange={v => setValue2(v)} />
    </View>;

    基础样式二

    const [value1, setValue1] = React.useState(true);
    const [value2, setValue2] = React.useState(false);
    

    <View>
      <SwitchButton
        value={value1}
        size={{ height: 18 }}
        theme={{ margin: 0 }}
        onValueChange={v => setValue1(v)}
        style={{ marginRight: 14 }}
      />
      <SwitchButton
        value={value2}
        size={{ height: 18 }}
        theme={{ margin: 0 }}
        onValueChange={v => setValue2(v)}
      />
    </View>;

    基础文本样式
