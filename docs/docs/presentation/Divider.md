## 代码演示

### 基础形式

```jsx
import { Divider } from 'tuya-panel-kit'

<Divider color="red" width={300} style={{ marginLeft: 30 }} />
```

### 块状形式展示

```jsx
import { Divider } from 'tuya-panel-kit'

<Divider
  color="#333"
  width={300}
  height={30}
  style={{ marginLeft: 30, marginTop: 10 }}
/>
```
